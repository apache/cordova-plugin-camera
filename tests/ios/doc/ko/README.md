# CDVCamera에 대 한 iOS 테스트

`Node.js` `코르도바` ios에서를 설치 해야.

코르도바-ios를 설치 하는 첫번째는:

    npm install
    

현재 폴더에....

# Xcode에서 테스트

  1. `CDVCameraTest.xcworkspace` 파일을 시작 합니다.
  2. 구성표 드롭 다운 메뉴에서 "CDVCameraLibTests"를 선택
  3. 클릭 하 고 `재생` 버튼에는 테스트를 실행 하려면 `공구 모양` 아이콘을 선택

# 명령줄에서 테스트

    npm test
