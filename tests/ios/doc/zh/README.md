# CDVCamera 的 iOS 測試

您需要安裝`node.js`拉`科爾多瓦 ios`中.

第一次安裝科爾多瓦 ios:

    npm install
    

在當前資料夾中。

# 從 Xcode 測試

  1. 啟動`CDVCameraTest.xcworkspace`檔。
  2. 從方案下拉式功能表中選擇"CDVCameraLibTests"
  3. 按一下並堅持`播放`按鈕，然後選擇要運行的測試的`扳手`圖示

# 從命令列測試

    npm test
